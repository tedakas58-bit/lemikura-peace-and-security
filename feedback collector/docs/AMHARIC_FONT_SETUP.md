# የአማርኛ ፊደል ማዋቀሪያ መመሪያ - Amharic Font Setup Guide

## Overview
This guide helps you properly display Amharic text in Excel reports generated by the Customer Satisfaction Survey system.

## የሚመከሩ ፊደሎች - Recommended Fonts

### Built-in System Fonts
- **Nyala** - Available on Windows systems
- **Ebrima** - Available on Windows 8+ systems  
- **Kefa** - Available on macOS systems
- **Noto Sans Ethiopic** - Available from Google Fonts

### Project Fonts (docs/fonts/geez-free/)
The following high-quality Amharic fonts are included in this project:

#### Professional Fonts
- **Adwa Regular/Bold** - Clean, professional appearance
- **Shiromeda Regular/Bold/SemiBold** - Modern, readable
- **Addis Sans** - Sans-serif style
- **GeezDigital V1** - Digital-friendly

#### Decorative Fonts
- **Tayitu** - Elegant style
- **Entoto** - Traditional appearance
- **Neteru** - Artistic style

## Excel Setup Instructions

### Method 1: Change Font in Excel
1. Open the exported Excel file
2. Select all cells (Ctrl+A)
3. Go to Home tab > Font dropdown
4. Choose one of the recommended fonts above
5. The Amharic text should now display correctly

### Method 2: Install Project Fonts
1. Navigate to `docs/fonts/geez-free/FreeToRedistribute/`
2. Choose a font folder (e.g., `AnbassaDesign/`)
3. Double-click on a `.ttf` or `.otf` file
4. Click "Install" button
5. Restart Excel
6. Use the newly installed font

## Troubleshooting

### Text Shows as Boxes or Question Marks
- **Cause**: No Amharic font is installed or selected
- **Solution**: Install and select an Amharic font from the list above

### Text Shows Incorrectly
- **Cause**: Wrong font encoding or old font version
- **Solution**: Try a different font from the recommended list

### Font Not Available in Excel
- **Cause**: Font not properly installed
- **Solution**: 
  1. Reinstall the font
  2. Restart Excel completely
  3. Check Windows Font settings

## Technical Notes

### Character Encoding
- Reports are exported with UTF-8 encoding
- Excel should automatically detect Amharic characters
- If issues persist, try opening with "UTF-8" encoding explicitly

### Font Recommendations by Use Case
- **Official Reports**: Nyala, Ebrima, Adwa
- **Presentations**: Shiromeda, Addis Sans
- **Print Documents**: Adwa, Shiromeda Serif
- **Digital Display**: GeezDigital, Ebrima

## Support

If you continue to have issues with Amharic text display:
1. Verify your system supports Unicode
2. Try multiple fonts from the recommended list
3. Ensure Excel is updated to a recent version
4. Contact system administrator for font installation help
