### рЇЋрѕ«рїђріГрЅ▒ріЋ рѕЏрѕхрігрІх

рЅаріаріФрЅБрЅбрІФрІі рѕЂріћрЅ│ рЇЋрѕ«рїђріГрЅ▒ріЋ рѕѕрѕЏрѕхрігрІх рЅарѕГріФрЅ│ ріарѕЏрѕФрї«рЅй ріарѕЅрІјрЅх:

#### ріарѕЏрѕФрїГ 1: live-server рѕўрїарЅђрѕЮ (рѕѕрѕЇрѕЏрЅх рІерѕџрѕўріерѕГ)
```bash
npm run dev
```
рІГрѕЁ рЅаhttp://localhost:3000 рѕІрІГ рЅарЅђрїЦрЅ│ ріЦріЋрІ░рїѕріЊ рѕўрїФріЋ рїІрѕГ рІерѕЇрѕЏрЅх ріарїѕрѕЇрїІрІГ рІГрїђрѕЮрѕФрѕЇ

#### ріарѕЏрѕФрїГ 2: http-server рѕўрїарЅђрѕЮ
```bash
npm start
```
рІГрѕЁ рЅаhttp://localhost:3000 рѕІрІГ рЅђрѕІрѕЇ HTTP ріарїѕрѕЇрїІрІГ рІГрїђрѕЮрѕФрѕЇ

#### ріарѕЏрѕФрїГ 3: serve рѕўрїарЅђрѕЮ
```bash
npm run serve
```
рІГрѕЁ рЅаhttp://localhost:3000 рѕІрІГ рІерѕЏрІГріЋрЅђрѕ│рЅђрѕх рЇІрІГрѕјрЅйріЋ рІФрїѕрѕѕрїЇрѕІрѕЇ

## ріарІ▓рѕх рЅБрѕЁрѕфрІФрЅх

### рІерІюріЊ ріЦріЊ рЅЦрѕјрїЇ рѕхрѕГрІЊрЅх

- рЅ░рїарЅЃрѕџрІјрЅй рІерЅЁрѕГрЅЦ рїірІю рІюріЊрІјрЅйріЋрЇБ рѕЏрѕхрЅ│рІѕрЅѓрІФрІјрЅйріЋ ріЦріЊ рЅЦрѕјрїјрЅйріЋ рѕЏрІерЅх рІГрЅйрѕІрѕЅ
- ріЦрІФріЋрІ│ріЋрІ▒ рІюріЊ рѕЮрѕхрѕЇрЇБ рѕЮрІхрЅЦрЇБ рЅђріЋ ріЦріЊ рІГрІўрЅх ріарѕѕрІЇ
- рЅ░рїарЅЃрѕџрІјрЅй рІюріЊрІјрЅйріЋ рѕўрІЇрІ░рІх рІГрЅйрѕІрѕЅ
- рЅ░рїарЅЃрѕџрІјрЅй рЅарІюріЊрІјрЅй рѕІрІГ ріарѕхрЅ░рІФрІерЅх рѕўрѕхрїарЅх рІГрЅйрѕІрѕЅ
- рІерѕЎрѕЅ рІГрІўрЅх рѕърІ│рѕЇ ріЦрІГрЅ│

### рІеріарѕхрЅ░рІ│рІ│рѕф рЇЊріљрѕЇ

рІеріарѕхрЅ░рІ│рІ│рѕф рЇЊріљрѕЇ рЅа `admin.html` рѕІрІГ рІГрїѕріЏрѕЇ

**рІерѕўрїЇрЅбрІФ рѕЮрѕхріГрѕГріљрЅХрЅй:**
- рІерЅ░рїарЅЃрѕџ рѕхрѕЮ: `admin`
- рІерІГрѕѕрЇЇ рЅЃрѕЇ: `admin123`

**рІеріарѕхрЅ░рІ│рІ│рѕф рЅБрѕЁрѕфрІФрЅх:**
- ріарІ▓рѕх рІюріЊрІјрЅйріЋ ріЦріЊ рЅЦрѕјрїјрЅйріЋ рІФріГрѕЅ
- ріљрЅБрѕГ рІюріЊрІјрЅйріЋ рІФрѕГрЅхрІЉ
- рІюріЊрІјрЅйріЋ рІГрѕ░рѕГрІЎ
- ріарѕхрЅ░рІФрІерЅХрЅйріЋ рІФрѕхрЅ░рІ│рІхрѕЕ
- рѕхрЅ│рЅ▓рѕхрЅ▓ріГрѕх рІГрѕўрѕЇріерЅ▒ (рїарЅЁрѕІрѕІ рІюріЊрІјрЅйрЇБ ріарѕхрЅ░рІФрІерЅХрЅйрЇБ рІѕрІ│рїЁріљрЅХрЅй)
- рІерІхрѕе-рїѕрїй рЅЁріЋрЅЦрѕ«рЅйріЋ рІФрІІрЅЁрѕЕ

**рІеріарѕхрЅ░рІ│рІ│рѕф рЇЊріљрѕЇ ріГрЇЇрѕјрЅй:**
1. **рІюріЊрІјрЅй ріЦріЊ рЅЦрѕјрїјрЅй:** рѕЂрѕЅріЋрѕЮ рІюріЊрІјрЅй рІФрѕхрЅ░рІ│рІхрѕЕ
2. **ріарѕхрЅ░рІФрІерЅХрЅй:** рѕЂрѕЅріЋрѕЮ рІерЅ░рїарЅЃрѕџ ріарѕхрЅ░рІФрІерЅХрЅйріЋ рІГрѕўрѕЇріерЅ▒ ріЦріЊ рІФрѕхрЅ░рІ│рІхрѕЕ
3. **рЅЁріЋрЅЦрѕ«рЅй:** рІерІхрѕе-рїѕрїй рѕўрѕерїЃ ріЦріЊ рІеріЦрІЇрЅѓрІФ рІЮрѕГрІЮрѕ«рЅйріЋ рІФрІІрЅЁрѕЕ

### рІерІЇрѕѓрЅЦ рѕЏріерѕЏрЅ╗

- рѕЂрѕЅрѕЮ рІ│рЅ│ рЅаріарѕ│рѕй localStorage рІЇрѕхрїЦ рЅ░ріерѕЏрЅйрЅирѕЇ
- рѕѕрѕЮрѕГрЅх ріарїарЅЃрЅђрѕЮрЇБ рІГрѕЁ рЅаріарїѕрѕЇрїІрІГ-рїјріЋ рІ│рЅ│рЅцрІЮ рѕўрЅ░ріФрЅх ріарѕѕрЅарЅх (MongoDBрЇБ MySQLрЇБ рІѕрІўрЅ░.)

## рѕЏрЅарїђрЅх

- **рЅђрѕѕрѕърЅй**: рЅа`style.css` рІЇрѕхрїЦ рЅа`:root` рѕЮрѕФрїГ рІЇрѕхрїЦ CSS рЅЦрїЂ рЅБрѕЁрѕфрІФрЅхріЋ рІФрѕхрЅ░ріФріГрѕЅ
- **рІГрІўрЅх**: рЅа`index.html` рІЇрѕхрїЦ HTML рІГрІўрЅхріЋ рІФрѕГрЅхрІЉ
- **рЅЁрїЦ**: рЅа`css/style.css` рІЇрѕхрїЦ рЅЁрїдрЅйріЋ рІФрЅарїЂ
- **рЅ░рїЇрЅБрѕГ**: рЅа`js/script.js` рІЇрѕхрїЦ JavaScript рІФріГрѕЅ рІѕрІГрѕЮ рІФрѕхрЅ░ріФріГрѕЅ
- **рІеріарѕхрЅ░рІ│рІ│рѕф рѕЮрѕхріГрѕГріљрЅХрЅй**: рЅа`js/admin.js` рІЇрѕхрїЦ рІерѕўрїЇрЅбрІФ рѕЮрѕхріГрѕГріљрЅХрЅйріЋ рІГрЅђрІГрѕЕ
